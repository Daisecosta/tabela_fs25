# Subindo o projeto para o Git
## projeto clonado

### 1. status atual antes de subir

```bash
git status
```

### 2. Adicionando os arquivos 

```bash
git add .
```

### 3. verificar o status do adicionar

```bash
git status
```

### 4. preparar para subir alterações

```bash
git commit -m "mensagem"
```

### 5. verificar o status do commit

```bash
git status
```

### 6. subir alterações

```bash
git push
```
